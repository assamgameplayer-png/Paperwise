# Contributing to PaperWise 🤝

First off — thank you for even thinking about contributing! This is a student project
and every bit of help means a lot.

## Ways to Contribute

- **Bug reports** — Found something broken? Open an Issue and describe what happened.
- **Feature ideas** — Have an idea that would help students? Open an Issue tagged `enhancement`.
- **Code** — Fork the repo, make your changes on a new branch, open a Pull Request.
- **Documentation** — Even fixing a typo in the README counts!

## Getting Started

```bash
git clone https://github.com/YOUR_USERNAME/paperwise.git
cd paperwise
pip install kivy==2.3.0 kivymd==1.2.0
python main.py
```

## Pull Request Guidelines

1. One feature or fix per PR — keeps things easy to review.
2. Write a short description of *why* you made the change, not just *what*.
3. Test that the app still runs on desktop before submitting.

## Code Style

- Follow PEP 8 (snake_case for functions and variables).
- Add a short comment above any block of logic that isn't obvious.
- Keep functions small — if a function does more than one thing, split it.

## Reporting Bugs

Please include:
- Your OS and Python version
- What you expected to happen
- What actually happened (paste the error if there is one)

---

This is an open, friendly project. Be kind to each other. 🙂
